一、什么是黑苹果?

在非苹果设备上运行 macos
原因我分为两类：

  1. 不够（或者不舍得）买苹果电脑的钱
  2. 单纯的喜欢折腾

目的有以下几条：

  • 体验 macos 系统下优秀的视频创作环境（非视频创作者，道听途说而已），例如 Final Cut Pro
  • 已有苹果的其他产品，想体验完整的苹果生态，随航、接力和”一响全响“的通知。。。。。。。
  • 作为主系统体验 unix 环境下的代码编辑工作或者 IOS 开发
  • 体验 macos 下漂亮的 UI 设计，喜欢 macos 下的流畅运行
